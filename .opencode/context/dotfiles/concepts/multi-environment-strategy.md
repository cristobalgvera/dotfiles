# Multi-Environment Strategy

## Summary

This repo targets **macOS** environments only, sharing a consistent baseline across personal machines.

## Principles

- Keep repo configs portable across macOS devices
- Use `~/.zshrc.local` for machine-specific overrides
- Avoid storing secrets in repo
- Prefer tooling that is consistent across macOS systems
